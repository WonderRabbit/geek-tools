# Wonder-write-r

Use the Rust-backed OpenCode tools for large writes and patch application.

- Use `fast_write_r` for large file creation or whole-file replacement.
- Use `fast_apply_r` for unified-diff patch application.
- Keep paths workspace-relative.
- Set `WONDER_WRITE_R_EXE` to the absolute executable path when the binary is not installed at `.opencode/bin/wonder-write-r.exe`.
